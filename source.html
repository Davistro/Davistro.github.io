Interfaz grafica con Python/Tkinter
Hola mundo.

Código

from tkinter import * root = Tk() root.title("Hola Mundo")
win = Label(root, text="Hola Mundo desde Python"'\n' "Usando Tkinter")
win.pack()
root.mainloop()

Resultado

Explicación

Lo primero que debemos hacer al igual que con otros módulos,
es que debemos importarlo para poder comenzar a utilizarlo.
1. from tkinter import *

Creamos la raiz de la aplicacion
3. root = Tk()

Le damos el titulo a la ventana
4. root.title("Hola Mundo)

Usamos una variable para mostrar el texto en este caso "win"
5. win = Label(root, text="Hola Mundo desde python" '\n' "Usando Tkinter)

Empaquetamos
6. win.pack()

Bucle de la app, o en otras palabrass para que la app no se cierre sola
7. root.mainloop()

Y listo, nuestro primer programa con Tkinter.
